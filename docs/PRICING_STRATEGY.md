# Estrategia de Precios y Costos - TechNova

Para definir precios rentables, primero debemos entender que no vendemos "páginas", vendemos **soluciones basadas en tiempo de talento y costos de infraestructura**.

## 1. Naturaleza de los Costos

*   **Costos Fijos (Infraestructura/SaaS):** Hosting (Vercel/AWS), Dominios, Licencias de software (ej. plantillas premium), tokens de API (OpenAI para chatbots), herramientas de automatización (Zapier). *Estos costos nunca los debe absorber la agencia a largo plazo; se cobran en un fee mensual o se delegan a la tarjeta del cliente.*
*   **Costos Variables (Talento / Horas):** Es el tiempo que le toma a cada integrante del equipo investigar, diseñar, programar y lanzar.

---

## 2. Desglose de Paquetes por Roles y Valor

### Paquete 1: START (Validación y Presencia / Landing Page)
**Entregable:** Sitio de una página enfocado en captación de leads + Setup básico.
*   **Research & Estrategia (20%):** Entender el modelo de negocio, definir la propuesta de valor y hacer el Copywriting persuasivo.
*   **UI/UX (30%):** Creación visual rápida (usando componentes pre-fabricados de nuestro Design System para ahorrar tiempo).
*   **Desarrollo Web (40%):** Programación frontend (React/Tailwind), formularios y despliegue (Vercel).
*   **SEO & Data Analytics (10%):** Instalación de Google Tag Manager, Meta Pixel y SEO On-Page básico.
*   **Modelo de Cobro Sugerido:** Precio Fijo por proyecto + Fee mínimo anual por hosting/dominio.

### Paquete 2: GROWTH (Escalabilidad / E-Commerce & CMS)
**Entregable:** Tienda online o plataforma autogestionable con automatizaciones.
*   **Research & Estrategia (15%):** Arquitectura de la información, estructuración del catálogo de productos.
*   **UI/UX (25%):** Diseño del flujo de compra (Checkout sin fricción), páginas de producto dinámicas.
*   **Desarrollo Web (45%):** Backend, bases de datos (Neon), integración de pasarelas de pago (Stripe) y carritos de compra.
*   **Marketing & SEO (15%):** Automatización de correos (carritos abandonados), SEO técnico avanzado.
*   **Modelo de Cobro Sugerido:** Setup Base (Desarrollo) + Retainer Mensual (Mantenimiento, soporte y actualizaciones).

### Paquete 3: SCALE (Operaciones / IA, CRM, Desarrollo a Medida)
**Entregable:** Agentes de IA 24/7, integraciones de CRM, software complejo.
*   **Data Analytics & Estrategia (30%):** Limpieza de datos del cliente para entrenar a la IA (RAG), diseño de flujos operativos (Pipelines de CRM).
*   **Desarrollo Web & Backend (50%):** Integración pesada de APIs, lógica de negocio compleja, infraestructura cloud de alto nivel.
*   **UI/UX (10%):** Dashboards administrativos internos.
*   **Marketing & Soporte (10%):** Capacitación al equipo de ventas del cliente para que adopten la tecnología.
*   **Modelo de Cobro Sugerido:** Setup Base Alto + Suscripción (SaaS) mensual por el uso de los Agentes IA, tokens y soporte continuo.

---

## 3. Resumen Estratégico
El valor más grande que entrega TechNova está en la intersección de **Desarrollo + Marketing**. No vendemos código, vendemos sistemas que traen clientes (Start/Growth) o que reducen costos operativos (Scale / Chatbots).
