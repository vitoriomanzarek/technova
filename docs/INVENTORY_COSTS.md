# Inventario de Servicios y Estructura de Costos (A La Carte) - TechNova

Para poder armar paquetes flexibles (Start, Growth, Scale) o vender soluciones especializadas, aquí está el desglose "A La Carte" de todos nuestros subproductos con su valor de mercado sugerido (Precios en USD).

## 1. Infraestructura (Pass-Through / Costos Fijos)
*Por lo general, estos se le cobran directo a la tarjeta del cliente o se facturan anualmente sin margen de ganancia, solo como gestión.*
*   **Dominio Web (.com, .net, etc.):** $15 - $25 / año.
*   **Hosting Básico (Landing Pages / Vercel):** $150 - $250 / año (O $0 si usamos tier gratuito).
*   **Hosting Avanzado (E-Commerce / Bases de Datos Neon/AWS):** $500 - $1,200 / año.
*   **Certificados de Seguridad SSL Avanzados:** $50 - $100 / año.

## 2. Identidad Visual y Diseño (UI/UX)
*   **Diseño de Logo Básico (Isotipo + Logotipo):** $250 - $500
*   **Branding Completo (Manual de identidad, colores, tipografía, aplicaciones):** $1,000 - $2,500
*   **Design System Completo (Librería de componentes React/Figma para apps escalables):** $1,500 - $3,500
*   **Diseño de Interfaz (UI) por pantalla (Web/App):** $100 - $250 / pantalla.

## 3. Estrategia y Marketing
*   **Research de Marketing (Auditoría, Buyer Persona, Competencia):** $800 - $1,500
*   **Copywriting Persuasivo (Redacción de Landing Page):** $400 - $800
*   **Setup de Campaña SEO (Auditoría técnica On-Page, optimización de velocidad):** $500 - $1,200
*   **Gestión SEO Mensual (Creación de artículos, Backlinks, Monitoreo):** $500 - $1,500 / mes.
*   **Gestión de Ads (Meta/Google):** $500 - $1,000 / mes (+ El Ad Spend que paga el cliente).

## 4. Desarrollo e Ingeniería Web
*   **Desarrollo de Landing Page (Frontend estático / UI Básica):** $800 - $1,200
*   **Desarrollo de E-Commerce (Catálogo, Carrito, Stripe, CMS):** $2,500 - $5,000+
*   **Integración de Pasarela de Pagos (Stripe / PayPal Custom):** $500 - $1,000
*   **Dashboard de Análisis de Datos (Panel BI / Gráficas conectadas a DB):** $1,500 - $4,000
*   **Setup de CRM Personalizado (Integración de APIs y flujos):** $1,500 - $3,000

## 5. Inteligencia Artificial y Automatización
*   **Desarrollo de Agente IA Básico (Reglas + FAQ):** $800 - $1,500
*   **Desarrollo de Agente IA Avanzado (RAG, entrenado con PDFs/Data del cliente):** $3,000 - $6,000+
*   **Suscripción Mensual de IA (Mantenimiento, Refinamiento y Límite de Tokens):** $500 - $2,000 / mes.
*   **Automatización de Flujos (Zapier/Make - Ej. Lead a WhatsApp a CRM):** $300 - $800 por flujo.

---

## ¿Cómo armar los Paquetes con esta lista?
Sabiendo esto, el **Paquete START ($1,000)** es en realidad la suma con descuento de:
*(Logo Básico $0 si ya tiene + Copywriting $400 + Desarrollo Landing $800 + Setup SEO $0 = Valor Real $1,200).*

El **Paquete GROWTH ($3,500)** suma:
*(Research $800 + Desarrollo E-Commerce $3,000 + Pasarela Pagos $500 = Valor Real $4,300).*
